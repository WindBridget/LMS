   Mục tiêu bài học

Mục tiêu bài học:

Bài học này giới thiệu những kiến thức nền tảng về lập trình hướng đối tượng. Bài học giúp người học biết khái niệm thế nào là Lớp (Class), Đối tượng (Object). Bài học cũng hướng dẫn người học thực hiện được việc trừu tượng hóa được các lớp đối tượng và khởi tạo được các đối tượng trong lập trình.

Chuẩn đầu ra:

PRO192x_o8 - Hiểu thế nào là trừu tượng hóa và thực hiện được trừu tượng hóa được lớp cơ bản (Class)

PRO192x_o9 - Phân biệt được thế nào là lớp thế nào là đối tượng (Class & Object)

PRO192x_o10 - Sử dụng được contructor và khởi tạo được các đối tượng.

Thời gian học:

2h

Chuẩn bị:

- Đã cài đặt môi trường làm việc với ngôn ngữ lập trình Java (JDK và một IDE)

- Đã cài đặt plugin cho trình duyệt Chrome FUNiX Passport để xem nội dung ở tiếng Việt.

QUIZ 5 (5/5 points)

1. Mệnh đề nào đúng nhất về đối tượng (Object)?

2. Xem xét đoạn code sau:

      1. public class Test1 {
      2. public float aMethod(float a, float b) {
      3. }
      4.
      5. }

Chọn những phương thức có thể thêm vào dòng số 4 của đoạn code trên? (Chọn 2)

3. Xem đoạn code sau:

    class Point {
      int m_x, m_y;

      public Point(int x, int y) { m_x = x; m_y = y; }
      public Point() { this(10, 10); }
      public int getX() { return m_x; }
      public int getY() { return m_y; }

      public static void main(String args[]) {
        Point p = new Point();
        System.out.println(p.getX());
      }
    }

4. Từ khóa nào làm cho biến thuộc một lớp mà không được xác định ở một thể hiện của lớp

5. Màn hình sẽ hiển thị kết quả bao nhiêu

    public class Frog {
    static int count=0;
    public Frog() { count++; }
    public static void main(String[] args)
      {
          new Frog();
          new Frog();
          new Frog();
          System.out.println("count=" + count);
      }
    }

Course Updates & News   

JUNE 1, 2016

Chào mừng các bạn đến với khóa học lập trình di động trên nền tảng Android!

Android đang là hệ điều hành chiếm lĩnh thị trường thiết bị động hiện nay. Hàng trăm ngàn thiết bị từ ti vi, máy nghe nhạc... cho đến điện thoại di động đều có thể chạy trên nền Android.

Với xu hướng ngày càng phát triển của Android, đã kéo theo nhu cầu tuyển dụng lập trình Android ngày càng khốc liệt. Và lập trình Android chính là cơ hội tuyệt vời nhất, đặc biệt là các bạn trẻ.

Nắm bắt được nhu cầu tuyển dụng lập trình Android ngày càng nóng, FUNiX xây dựng chương trình PRM391 giúp học viên sẽ đủ khả năng ứng tuyển vị trí lập trình Android tại các công ty. hoặc trở thành một Freelancer với mức lương tự do.

Để bắt đầu, hãy xem mục Courseware để đọc các nội dung tổng quan của khóa học ở Phần Giới thiệu. Phần giới thiệu sẽ giúp bạn có cái nhìn tổng quan về nội dung khóa học, cấu trúc, lịch trình và cách thức học khóa này.

Ngoài ra, chúng tôi hiểu bạn có thể gặp phải những lỗi kỹ thuật do sự khác biệt về các phần mềm sử dụng. Do đó chúng tôi khuyến khích bạn phản hồi cả về vấn đề kỹ thuật và nội dung trong xRoom nếu thấy có bất cứ điều gì khiến bạn bối rối hoặc không hoạt động như ý bạn mong muốn.

Một lần nữa, chào mừng và cám ơn bạn đã tham gia khóa học. Chúc bạn đạt được kết quả tốt nhất!
